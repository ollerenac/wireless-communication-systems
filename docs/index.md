## Descripción

Este curso ofrece un tratamiento riguroso y actualizado de los sistemas de comunicaciones inalámbricas. Cada sesión combina teoría matemática con un laboratorio práctico en Python ejecutable directamente en Google Colab.

El curso cubre tres arcos temáticos:

1. **Fundamentos de Capa Física** (Sesiones 1–4): modelado de canal, modulación, OFDM, codificación de canal
2. **Sistemas Multi-Antena y PHY Avanzada** (Sesiones 5–7): MIMO, MIMO masivo, estimación de canal
3. **IA/ML para Sistemas Inalámbricos y Tecnologías Emergentes** (Sesiones 8–15): aprendizaje automático, aprendizaje profundo, RL, RIS, ISAC, 6G y redes especializadas
