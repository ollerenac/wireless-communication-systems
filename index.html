    Sistemas de Comunicaciones Inalámbricas

    Curso de Máster · Acceso Abierto · CC BY 4.0

    Un tratamiento riguroso y moderno de las comunicaciones inalámbricas — desde los fundamentos de la capa física hasta el diseño de sistemas 5G/6G con inteligencia artificial. Cada sesión combina teoría matemática con un laboratorio Python ejecutable directamente en Google Colab.

    Ver programa completo Repositorio GitHub

    Arcos Temáticos

    • Fundamentos de Capa Física

      Sesiones 1–5

      Modelado de canal inalámbrico, modulación digital y BER, sistemas OFDM, codificación de canal con LDPC y códigos Polares, acceso múltiple: OFDMA y NOMA.

    • Sistemas Multi-Antena

      Sesiones 6–8

      Fundamentos MIMO y capacidad, MIMO masivo con conformación de haz digital, estimación de canal clásica.

    • Tecnologías Avanzadas y Emergentes

      Sesiones 9–15

      5G NR, mmWave y sub-THz, RIS, ISAC, radio cognitiva y gestión de espectro, IA/ML para comunicaciones inalámbricas, fronteras del 6G.

    Prerrequisitos

    Para aprovechar al máximo el curso, el estudiante debe tener conocimientos de nivel de grado en:

    Área Contenidos clave
    Señales y Sistemas Transformada de Fourier, convolución, análisis en frecuencia
    Probabilidad y Estadística Variables aleatorias, distribuciones, ruido gaussiano
    Comunicaciones Digitales Modulación en banda base, filtro adaptado, SNR
    Álgebra Lineal Operaciones matriciales, eigendecomposición, SVD
    Programación Python básico (NumPy, Matplotlib), cuadernos Jupyter

    Cómo Usar Este Curso

    Navega las sesiones directamente desde este sitio. Cada página incluye apuntes de teoría, ejercicios con soluciones desplegables y un botón Abrir en Colab para ejecutar el laboratorio sin instalar nada.

    git clone https://github.com/ollerenac/wireless-communication-systems.git
cd wireless-communication-systems
pip install -r requirements.txt
mkdocs serve    # vista previa en http://127.0.0.1:8000

    Cada sesión incluye un cuaderno Jupyter con un botón Open in Colab. Haz clic y el laboratorio se abre directamente en el navegador — sin instalaciones, sin configuración.
