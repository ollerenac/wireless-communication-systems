# Sistemas de Comunicaciones Inalámbricas

Curso de máster de acceso abierto sobre sistemas de comunicaciones inalámbricas — desde fundamentos de capa física hasta técnicas de 5G/6G e IA aplicada.

**Sitio web**: [ollerenac.github.io/wireless-communication-systems](https://ollerenac.github.io/wireless-communication-systems/)
**Licencia**: [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/)

---

## Contenido

15 sesiones semanales, cada una con apuntes de teoría y un laboratorio Python ejecutable en Google Colab:

| # | Título |
|---|--------|
| 01 | Modelado del Canal Inalámbrico |
| 02 | Modulación Digital y Análisis de BER |
| 03 | Sistemas OFDM e Implementación |
| 04 | Codificación de Canal: LDPC y Códigos Polares |
| 05 | Fundamentos MIMO y Capacidad |
| 06 | MIMO Masivo y Conformación de Haz Digital |
| 07 | Estimación de Canal: Clásica y Basada en Aprendizaje |
| 08 | Aprendizaje Automático para Redes Inalámbricas |
| 09 | Comunicaciones Aprendidas Extremo a Extremo |
| 10 | Aprendizaje Profundo para Detección MIMO |
| 11 | Aprendizaje por Refuerzo para Asignación de Recursos |
| 12 | Superficies Inteligentes Reconfigurables (RIS) |
| 13 | Detección e Integración con Comunicaciones (ISAC) |
| 14 | Fronteras del 6G: THz, MIMO Libre de Celdas y Aprendizaje Federado |
| 15 | Sistemas Inalámbricos Especializados |

## Uso local

```bash
git clone https://github.com/ollerenac/wireless-communication-systems.git
cd wireless-communication-systems
pip install -r requirements.txt
mkdocs serve    # vista previa en http://127.0.0.1:8000
```
